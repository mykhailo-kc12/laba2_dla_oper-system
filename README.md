# laba2_dla_oper-system

Лабораторна робота №2 з дисципліни "Операційні системи".

## Опис
Проєкт містить дві утиліти на Python:
- визначення шляху до конфігураційного файлу з урахуванням пріоритету аргументів командного рядка, змінних оточення та значення за замовчуванням;
- збір інформації про систему за допомогою опцій командного рядка.

## Функціональність
1. Отримання шляху до конфігураційного файлу:
   - `-c`, `--config`
   - `CONFIG_PATH`
   - `~/.config.yaml`

2. Отримання інформації про систему:
   - назва ОС
   - версія ядра
   - архітектура процесора
   - кількість логічних ядер
   - можливість збереження результату у файл

## Тестування
Для перевірки коректності роботи використано `pytest`.

## Перевірка стилю
Для перевірки якості коду використано `flake8`.
